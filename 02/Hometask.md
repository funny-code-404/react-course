# State and event handling

1. Перейдите в ветку `master`.
2. Обновите проект (upstream).
3. Создайте ветку `02` из ветки `master`.
4. Убедитесь, что вы находитесь в ветке `02`.
5. Используя форму из вашей верстки, сделайте поиск отелей:
   - поиск должен работать на основе данных из [массива](../lesson-3/array.js);
   - вам нужно работать только с полем "Your destination or hotel name";
   - результат поиска необходимо вывести в блок "Available hotels", который вы добавите сразу после блока "Top-section";
   - дизайн блока "Available hotels" идентичен блоку "Homes guests loves", только имеет заголовок "Available hotels".
6. Сделайте `commit` с комментарием, описывающим проделанную вами работу.
7. Сделайте `push` и создайте Pull request.
